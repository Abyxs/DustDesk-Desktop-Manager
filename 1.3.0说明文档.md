# DustDesk v1.3.0 版本说明

## 新增
- 新增剪贴板功能，支持记录文字和图片复制历史，并可（双击快捷）复制、删除和全部清除。
- 设置中心的桌面组件显示和快捷键目标新增“便签”选项。
- 新增桌面组件“锁定位置”，锁定后不可拖动和缩放，避免误操作。
- 支持将文件夹直接拖拽到桌面收纳组件中。
- 桌面组件快捷键支持勾选多个组件，并统一显示或隐藏。

## 优化
- 侧边栏顺序调整，设置中心固定放在最底部。
- 系统检测移除高占用进程显示，保留更实用的网络、内存、CPU、磁盘、延迟和运行时长信息。
- 主窗口快捷键改为打开/关闭切换。
- 锁定位置菜单增加锁定/未锁定图标，并在锁定后显示“已锁定”。
- 收纳组件拖拽文件/文件夹时，同名目标自动生成序号，避免覆盖原内容。


## 修复
- 主窗口右下角缩放体验优化，并降低最小窗口尺寸。
- 修复设置中心选项换行后显示不全、遮挡后续内容的问题。
- 修复从收纳组件拖回桌面时可能重复生成文件或文件夹的问题。

## 下载使用方法
- 在 GitHub Releases 页面下载 `DustDesk.1.3.0.zip`。
- 下载后解压到本地文件夹，例如 `D:\APP\DustDesk`。
- 双击运行 `DustDesk.exe` 即可使用。
- 本版本为 Windows x64 自包含版，不需要额外安装 .NET 运行库。
- 建议不要直接在压缩包内运行程序，请先完整解压。

## 升级后沿用之前配置
- 如果旧版本使用默认数据位置，新版本首次启动会自动读取原来的配置和数据。
- 如果旧版本使用过自定义数据位置，打开 `设置中心`，在 `文件保存路径` 点击 `选择`。
- 选择旧版本的数据文件夹后，程序会提示读取该位置的数据并重启。
- 选择读取后，原来的桌面收纳、工作记录、便签、项目、快捷启动、剪贴板和桌面组件设置会继续沿用。
- 升级前建议先备份旧版本的 `Data` 文件夹。
