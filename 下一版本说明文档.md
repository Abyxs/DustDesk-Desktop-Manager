# DustDesk 1.7.0 版本说明

## 更新调整

- 桌面收纳遇到同名文件或文件夹时改为直接拒绝移动，不再覆盖、不再自动改名。
- 桌面收纳组件拖出到桌面时先由应用处理移动，避免 Windows 桌面提前接收文件导致误判同名。
- 桌面收纳组件右键菜单改为应用内菜单，减少系统右键拖拽过程中的卡死风险。

## 修复问题

- 修复从收纳组件拖出文件后又回到组件内、无法落到桌面的问题。
- 修复桌面明明没有目标文件却弹出重名提示的问题。
- 修复拖拽流程卡住后桌面图标点击、右键无响应的问题。
- 修复收纳分类删除时物理目录残留导致内容状态不一致的问题。
- 补强快捷方式图标解析，支持 AppUserModelID 和 MSIX 应用图标回退。

## 新增功能

- 便签桌面组件支持折叠显示。

## 打包信息

- 版本号更新为 `1.7.0`。
- 当前为 1.7.0 修复版本，建议打包为 `DustDesk.1.7.0.zip`。
